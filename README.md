## ğŸ› ï¸ Dapur Teknis (The Tech Stack)

Walaupun ini website promosi, sebagai mahasiswa Fullstack, saya tetap memastikan kodenya rapi dan performanya ngebut.

* **Core:** HTML5 murni (Semantic).
* **Styling:** **Tailwind CSS** (via CDN). Saya memilih Tailwind karena fleksibilitasnya luar biasa untuk mengatur layout responsif tanpa harus "berkelahi" dengan file CSS terpisah yang menumpuk.
* **Animations:**
    * **AOS (Animate On Scroll):** Agar elemen-elemen muncul perlahan (fade-up/zoom-in) saat pengunjung menggulir halaman. Memberikan kesan elegan.
    * **Typed.js:** Efek ketikan di header ("Future Leader", "Global Innovator") untuk menarik perhatian sejak detik pertama.
* **Icons:** FontAwesome, karena ikon visual berbicara lebih cepat daripada teks.
